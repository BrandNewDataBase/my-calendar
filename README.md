# My Calendar 🗓️

따뜻한 팔레트의 나만의 캘린더 웹앱. 빌드 도구 없이 순수 HTML/CSS/JS(ES 모듈)로 만들어져
GitHub Pages에서 바로 동작합니다.

**▶ 사용하기: https://brandnewdatabase.github.io/my-calendar/**

## 기능

- **월 / 주 / 일 보기** — 드래그로 일정 생성·이동·리사이즈 (15분 스냅), 멀티데이 이벤트 레인, +N 더보기
- **일정 관리** — 카테고리 색상 8종, 반복 일정(매일/매주/매월/매년, 예외일, "이 일정만/모든 일정"), 검색, 실행취소
- **날짜 클릭 → 데이 패널** — 그 날의 일정 목록 + 음력 날짜 + 휴일 지정
- **음력 표시** — 모든 날짜에 음력 병기 (korean-lunar-calendar, 1000–2050년)
- **사용자 지정 휴일** — 원하는 날짜를 휴일로 지정하면 주황색으로 표시 (일요일 빨강 / 토요일 파랑)
- **알림** — 일정 하루 전 + 15분 전 브라우저 알림 (탭이 열려 있는 동안 동작, 권한 거부 시 인앱 토스트)
- **메모 탭** — 팔레트색 스티키 스타일 메모 카드, 고정/검색/8색
- **D-Day 카운터** — 헤더에 작게 표시
- **4개 언어** — 한국어 · English · 中文 · Français
- **라이트 / 다크 모드** — 따뜻한 크림 & 웜 차콜 팔레트
- **JSON 백업/복원** — 모든 데이터는 브라우저 localStorage에만 저장 (서버 없음)

## 키보드 단축키

| 키 | 동작 |
|----|------|
| `C` | 새 일정 |
| `T` | 오늘로 이동 |
| `M` / `W` / `D` | 월 / 주 / 일 보기 |
| `←` / `→` | 이전 / 다음 |
| `/` | 검색 |
| `Esc` | 패널/모달 닫기 |

## 로컬 실행

```bash
python -m http.server 8000
# http://localhost:8000 접속
```

## 개인정보

모든 데이터(일정, 메모, 설정)는 사용자의 브라우저 localStorage에만 저장되며
어떤 서버로도 전송되지 않습니다.
